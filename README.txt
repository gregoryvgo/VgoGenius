ADRAHTI — Υφάσματα & Διακόσμηση
Next.js 14 + Tailwind CSS Project

Βήματα:
1️⃣ npm install
2️⃣ npm run dev
3️⃣ Άνοιξε http://localhost:3000

Φάκελοι:
- app/ : κύριες σελίδες
- components/ : Header, Footer, Hero, Slider, ContactForm
- public/images/ : όλα τα media assets
- styles/globals.css : κεντρικά styles
